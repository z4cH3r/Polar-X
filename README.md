# What is Polar X?
Polar X is a index page designed for Gmod Communities.

# Whats Included?
- Portal Buttons.
- Config File.
- Server list w/ stats & connect button.
- Features Showcase.
- Staff Showcase.
- Social Media Showcase.
- Server statistics.
